double  _factor 
)

Apply AIE algorithm.

Parameters
_src: Source image to apply AIE algorithm (cv::Mat with CV_8UC1)
_dst: AIE applied Output image (cv::Mat with CV_8UC1)
_factor: Edge enhancement factor. As the _factor value is set higher, edge will be more enhanced but image will be more noisier. It should be greater than 0.
Remarks
This function applies AIE algorithm to enhance edge in _src and returns output image in _dst. This function get 8bit gray image, so 16bit image data which is received from TE should be converted to 8bit before apply AIE. If _src is empty or _factor is smaller than or equal to 0, it throws exception with type std::invalid_argument.

Image enhancement.

Parameters
input: 16bit input image to apply image enhancement
result: 16bit output image
height: height of image
width: width of image
filter_size: size of denoise filter. actual denoise filter size will be (2*filter_size) x (2*filter_size)
denoise_factor: factor to determine how much noise will be removed. Greater denoise_factor will suppress more noise, but image will be more blurred. typical range is 0~1000.
lamda: This parameter will determine how much image will be enhanced. large value of lamda can make image more noisy. typical range of this parameter is 0~10.
Remarks
This function sharpen edge of image and enhance image contrast. Since image enhancement algorithm also enhance noise, denoise algorithm is applied. In order to get clear image, proper setting of denoise and enhancement factor needs to be applied. Otherwise, image can be too blur or too noisy.
